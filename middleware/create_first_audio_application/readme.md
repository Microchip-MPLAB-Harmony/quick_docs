# Creating Your First Audio Application from Scratch

This tutorial provides information on how to create an audio project using MPLAB Harmony 3 and the <a href="https://www.microchip.com/Developmenttools/ProductDetails/DM320113" target="_blank">SAM E70 Xplained Ultra board</a>. It starts with a blank MPLABÂ® project and finishes with an audio application that is a subset of the audio_tone application.

To keep the application as simple as possible, no graphics or other UI are included. It simply outputs a 1000 Hz sine wave over the I2S interface at 48,000 samples/sec. The sine tone is generated internally using math functions (no audio inputs).
